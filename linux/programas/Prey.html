Instalei o PREY.
http://preyproject.com

O que faz?
Monitora o computador em caso de roubo.

Como foi instalado?
Via pacote debian disponivel no proprio site.
Usado o gerenciador grafico de pacotes debian.

Criei uma conta no site.
Na area do usuario existe um ID.
Na area dos dispositivos existe outro ID.
Ambos ids necessarios no arquivo "config".

Comando para teste (check)
./prey.sh --check

Diretorios/arquivos:
/usr/share/prey/prey.sh
/var/log/prey.log


Crontab setado para rodar o programa nos minutos: 0/20/40
dicas: crontab -l (lista o crontab do usuario)
crontab -e (edita o arquivo do crontab, text file)
