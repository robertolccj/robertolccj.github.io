Exemplos de arquivos
=====================
Voce pode ir direto no 'help / Exemples' na gui (interface) do xmgrace.

 OU
 
Muitos exemplos podem ser encontrados em:
  /usr/share/grace/examples/

Você poderá recorre a este exemplos se, por exemplo, não souber como
escrever um sobrescrito para colocar um índice em uma letra.


Layout Personalizado
=====================

Para se ter gráficos com layout padronizado com suas necessidades, voce pode
copiar o arquivo padrão que se localiza em:

 /usr/share/grace/templetes/Default.agr

Basta criar o diretorio abaixo com este arquivo:
 ~/.grace/templetes/Default.agr



Configurando modo de impressão padrão
======================================

Criar o arquivo:
 ~/.grace/gracerc.user

Adicionar as seguintes linhas:

HARDCOPY DEVICE "PostScipt"
DEVICE "PostScript" DPI 300


Alterando cores padrao
=======================

Edite o arquivo Defaults.agr, ou o arquivo que você está trabalhando.
Vá nas linhas parecida com:

  @map color 7 to (220, 220, 220), "grey"

Edite com as cores que você desejar, incusive renomeie!


Histogramas
============

Quando for fazer algumas transformações 'data / transformation / histograms',
por exemplo, deve-se selecionar uma 'source', mas não selecione a 'destination'.
