Instalando o zotero
======================
08/12/2012


Baixar no site:
 - http://www.zotero.org/

Pacote baixado, versao para linux 64bits:
 - Zotero-3.0.8_linux-x86_64.tar.bz2

Copiar para o diretorio: /opt

Extrair com: tar xvfj Zotero-XXX.tar.bz2

Criar um bash script para executar:
 - /usr/local/bin/zotero

Sugestao de script
#!/bin/bash
/opt/Zotero-XXX/zotero $@

obs1: cria diretorio de configuracoes em ~/.zotero
obs2: baixar estilo ABNT no proprio site do zotero,
      depois adicionar em 'preferencias' do zotero.
      (nao precisa o arquivo estar em diretorio preferencial)
