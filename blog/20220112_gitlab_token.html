# Modificando a segurança no Gitlab *Data: 12/01/2022* Hoje modifiquei a forma de logar no Github. Ativei a 2FA (autenticação em dois fatores). Salvei no KeePassXC no meu laptop. Quando fui atualizar a pasta (projeto) via `git push gitlab` não consegui. Tive que gerar um *Token* no site do gitlab e modificar o arquivo `.git/config` no meu pc. Peguei a dica em [stackoverflow](https://stackoverflow.com/a/51691209) via [dev.to](https://dev.to/fpeluso/how-to-clone-a-gitlab-repository-after-enabling-2fa-6oc).
