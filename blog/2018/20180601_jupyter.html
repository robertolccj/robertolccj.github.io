# Acessando o arquivo Jupyter O objetivo aqui é apenas fazer um teste. Vamos observar se é possível disponibilizar um aquivo Jupyter no GitHub, e visualizá-lo compilado. Testando: [clique aqui](https://nbviewer.ipython.org/github/robertolccj/robertolccj.github.io/blob/master/blog/2018/fisica-matematica-00.ipynb) Veja que é possível modificar e executar o código sem necessidade de ter o Jupyter no computador. Acessando o link anterior, é só clicando na opção `Execute on Binder`.
