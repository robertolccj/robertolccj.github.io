#### Customiza as margens da folha ```tex \usepackage[ top=1.5cm, bottom=1.5cm, left=1.5cm, right=1.5cm, headsep=10pt, a4paper] {geometry} ``` #### Não quebrar equação matemática Inserir chaves dentro do ambiante matemático 'em linha' (inline). ``` ${x=y+2}$ ``` #### Ambiente *itemize*: retira espaço à esquerda ```tex \usepackage{enumitem} \begin{itemize}[leftmargin=*] \item Esfera metálica. \end{itemize} ``` #### Inserir linha para escrita (horizontal) ```tex \line(1,0){470} \par ``` #### Começar texto em uma página ìmpar ```tex \cleardoublepage \newpage ``` #### Inserir espaçamento horizontal ```tex \vfill %preenche completamente o espaço entre dois parágrafos. \vspace{1cm} %preenche apenas 1cm ``` #### Figura: posicionamento forçado Para obrigar posicionamento da figura: usar *H* ```tex \usepackage{float} \begin{figure}[H] \includegraphics[scale=0.45]{figs/circuito01.png} \end{figure} ``` #### Tabela: altura da célula/linhas ```tex \begin{table} \renewcommand*\arraystretch{1.25} %Altura das linhas \begin{tabular}{|c|c|c|} . . \end{tabular} \end{table} ``` #### Caption: figuras e tabelas Inserir código no preâmbulo: ```tex \setlength{\abovecaptionskip}{5mm} \setlength{\belowcaptionskip}{2mm} ``` #### Ambiente *minipage* Uma minipágina de 40% da largura do texto (linha de texto). ```tex \begin{minipage}[t]{0.48\textwidth} Texto dentro da mini página. \end{minipage} ``` #### Combilar com o *biber* O biber aceita caracteres utf8 (é o furturo!) ```tex \usepackage[ style=numeric,
